# 📝 To-Do List Web App

A simple and responsive To-Do List made using **HTML, CSS, and JavaScript**.  
It supports:

- ✅ Adding tasks with date and time
- ✅ Marking tasks as completed
- ✅ Deleting tasks
- 💡 Clean and dark-themed interface
